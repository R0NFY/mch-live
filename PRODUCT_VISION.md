# Product Vision

The goal is to create a high-fidelity, low-latency AI voice assistant with a visual avatar. 

### Key Features:
- **Multilingual Support**: Primary focus on Russian language.
- **Realistic Avatar**: Integration with Simli for visual presence.
- **Natural Interaction**: Low-latency responses using LiveKit's real-time pipeline.
- **Post-Call Data retrieval**: Capture and store call transcripts, audio, and video for later analysis or sharing.
